_example processing rules file_	
	executed for every line
	parser inserts only first group
	log = table name
	level = log table's column (VARCHAR(5000))
{{{ log.class=\[([\w\.]+)\] log.level=^[\d-\s:,]+(\w+) log.time=(^[\d-\s:,]+) log.data=\[[\w\.]+\]\s(.+) }}}
